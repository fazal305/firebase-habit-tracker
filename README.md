# Firebase Habit Tracker

A real-time habit tracking web app built with Firebase Authentication, Firestore, Bootstrap 5, jQuery, and Vanilla JavaScript.

Users can sign up, sign in, create habits, mark daily completions, track streaks, and keep their progress synced in the cloud.

## Features

- User sign up and sign in with Firebase Authentication
- Per-user habit storage using Firestore
- Add, edit, and delete habits
- Mark habits complete or incomplete for the current day
- Current streak tracking
- Weekly completion progress
- Real-time UI updates with Firestore listeners
- Dark neon responsive interface
- Mobile-friendly layout

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- jQuery
- Bootstrap 5
- Firebase Authentication
- Firebase Firestore
